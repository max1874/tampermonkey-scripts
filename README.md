## 安装方法

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击对应脚本的 `.user.js` 文件
3. 点击 "Raw" 按钮，Tampermonkey 会自动提示安装

## 目录结构

```
tampermonkey-scripts/
├── scripts/          # 所有脚本
│   ├── script-1/
│   └── script-2/
└── README.md
```

## 开发说明

每个脚本都在 `scripts/` 目录下有独立的文件夹，包含：
- `*.user.js` - 油猴脚本文件
- `README.md` - 脚本说明（可选）
